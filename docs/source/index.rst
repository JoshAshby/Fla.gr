Welcome to fla.gr's documentation!
==================================

fla.gr is a side project of JoshAshby which aims to provide a good place to
learn and hack together vaious bits of Python, Javascript/Coffeescript and
possibly other languages. It's current incanation is that of a bookmarking
service such as delicious, offering the ability to bookmark a page, and add a
description and tags. 

This documentation aims to provide a learning ground for getting documentation
into the various modules that are used throughout fla.gr, both in Python and
other languages too.
